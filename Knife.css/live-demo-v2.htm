Let's cut them down!

Knife.css is a light-weight (<8kb minified), pure css based typography library to play with text and create interactive animations and micro-interactions. It features almost 18 types of (sharp!) knifes that are ready to cut any text as well as icons*. This is the showcase of that. Pick any of the knifes to see its usage and source code. Hover the text to see the effect.
